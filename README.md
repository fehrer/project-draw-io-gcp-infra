# project-draw-io-gcp-infra
Projeto de modelagem organizacional com segregação de departamentos (Helpdesk e Financeiro), sistemas (GLPI e ERP) e ambientes (DEV e PROD). Aplicação de RBAC com grupos de acesso específicos por sistema e ambiente, seguindo o princípio do menor privilégio e boas práticas de segurança e governança.
